# 📚 Bookia - Book Store App

A full-featured book store mobile application built with Flutter.

---

## ✨ Features

- **Authentication** — Login, Register, Forgot Password, OTP Verification, Reset Password
- **Home** — Browse books, Best Sellers, New Arrivals, Sliders
- **Book Details** — View full book info and add to cart
- **Search** — Search for books by name
- **Wishlist** — Save favorite books
- **Cart** — Manage cart items and quantities
- **Place Order** — Checkout with governorate selection
- **Order History** — View all previous orders
- **Profile** — View and edit profile info
- **Update Password** — Change account password

---

## 🛠 Tech Stack

- **Flutter** — UI Framework
- **Dart** — Programming Language
- **Flutter Bloc / Cubit** — State Management
